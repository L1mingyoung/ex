- **容器化编排**：利用 `docker-compose.yml` 定义服务拓扑，强制规定 `api` 服务必须在 `postgres` 和 `embedding` 服务健康检查通过后启动。
- **跨语言通信契约**：后端通过环境变量 `PYTHON_EMBED_URL` 动态发现并调用 Python 向量服务；前端与适配器通过共享的 `shared/types.ts` 确保 API 响应结构与状态管理的一致性。
- **多端适配架构**：在根目录维护 `adapters/` 目录，将 QQ Bot 等外部渠道作为独立进程或容器挂载，通过内部网络指向核心 API 服务，实现业务逻辑与接入层的解耦。
