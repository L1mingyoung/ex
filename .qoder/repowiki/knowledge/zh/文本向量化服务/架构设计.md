- 采用分层架构：`main.py` 作为 API 入口层，处理 HTTP 请求与响应；`embedder.py` 作为核心推理层，封装 Tokenizer 预处理、ONNX 模型推理及 Mean Pooling 后处理逻辑。
- 支持 Mock 模式：通过 `--mock` 参数或环境变量启用假向量生成，便于在无模型文件时调试接口流程。
- 依赖外部脚本：通过 `scripts/download_model.py` 从 Hugging Face Hub 下载并本地化存储 Jina 中文 Embedding 模型及 Tokenizer 文件。
- 配置灵活：支持通过环境变量自定义模型路径、Tokenizer 路径及最大序列长度。
