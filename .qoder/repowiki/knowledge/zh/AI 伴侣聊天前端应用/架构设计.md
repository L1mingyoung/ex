- **技术栈**: 采用 React 19 + TypeScript + Vite 6.0，通过 `@vitejs/plugin-react` 进行编译。
- **状态管理**: 使用 Context API (`AppContext.tsx`) 结合 `useReducer` 实现全局状态管理，涵盖角色、会话、消息及流式传输状态，并通过自定义 Hook `useAppContext` 暴露业务逻辑。
- **组件架构**: 采用复合组件模式，核心布局由 `Sidebar`（侧边栏）和 `ChatArea`（聊天区）组成；`ChatArea` 进一步拆分为 `ChatHeader`、`MessageList` 和 `InputArea`。
- **API 层**: `src/api/index.ts` 封装了 RESTful 请求与 SSE 流式通信逻辑，支持断点续传与错误处理，并与后端 NestJS 服务通过 Vite 代理交互。
- **样式与主题**: 使用 CSS Modules/Global CSS 配合 `data-theme` 属性实现深色/浅色主题切换，主题状态持久化至 `localStorage`。
