- **技术栈**: 采用 React 19 + TypeScript + Vite 6.0，使用 `@vitejs/plugin-react` 进行编译。
- **状态管理**: 核心逻辑集中在 `src/context/AppContext.tsx`，通过 `useReducer` 管理全局状态（角色、会话、消息流），并提供封装好的业务操作函数（如 `sendMessage`, `loadCharacters`）。
- **组件架构**: 遵循容器-展示分离模式。`App.tsx` 作为根容器组合 `Sidebar`（侧边栏）和 `ChatArea`（聊天区）两大布局模块。各模块内部进一步拆分为细粒度组件（如 `MessageBubble`, `CharacterItem`）。
- **API 抽象**: `src/api/index.ts` 封装了所有后端 HTTP 请求及 SSE 流式处理逻辑，支持断点续传与取消请求，并与 UI 层解耦。
- **样式与主题**: 使用 CSS Modules/局部 CSS 文件配合 `data-theme` 属性实现深色/浅色主题切换，由 `useTheme` Hook 统一管理。
